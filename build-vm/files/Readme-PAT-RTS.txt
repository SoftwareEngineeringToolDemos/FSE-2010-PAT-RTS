This file contains instructions on how to use Process Analysis Toolkit.

1. Start Process Analysis Toolkit using the "PAT 3" icon on your desktop. PAT can also be started from its default installation location - "C:\Program Files\Process Analysis Toolkit\Process Analysis Toolkit 3.5.1".

2. The tool should start up with the New Template Wizard being shown. Choose "Real-Time System Model" from the modeling languages panel to the left (or choose a different model if you so wish), and choose an assertion of your choice from the Templates panel to the right. Press "Ok" to start editing the newly created template.

3. Alternatively, you can can press "Cancel" to launch the tool without a new template.
